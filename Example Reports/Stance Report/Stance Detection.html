Data Description

Stance Detection computes Pro/Con stances for nodes in a primary nodeset and for nodesets linked to them. Initial 'seed' stances for the Primary nodes are specified directly or derived from neighbor nodes. Primary node stances are then propagated across interaction networks.

Social media data with document nodes, a document authorship network, and document x concept networks can use the Document label propagation algorithm to improve results.

Additional networks used to compute stances: Tweet x Hashtag, Tweet x Url
Algorithm: General label propagation
