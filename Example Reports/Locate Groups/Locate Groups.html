Group Sub-Network Measures

This section computes and displays information about the largest 100 groups (with at least 3 nodes). Measure calculations are performed on the sub-network defined by the group, e.g. the group member nodes and the links between them. The internal link count is the number of links that connect one group node to another group node. The external link count is the number of links that connect a group node to a non-group node. The External/Internal index ranges between -1 (all links are internal and the group is perfectly silo'd) to +1 (all links are external); a score of 0 means an equal number of internal and external links. The Cheeger value is the ratio of the group's external link count to its size. The approximate Cheeger value for the graph is the minimum value across all sub-groups.
